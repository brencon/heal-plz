# Debug Issue

Investigate and diagnose the following issue: $ARGUMENTS

## Investigation Steps

1. **Reproduce**: Understand how to trigger the issue
2. **Locate**: Search for relevant code paths and error sources
3. **Analyze**: Use git history to understand recent changes
4. **Trace**: Follow the execution flow to identify the root cause
5. **Propose**: Suggest a fix with explanation

## Required Information

- Error messages and stack traces
- Steps to reproduce
- Expected vs actual behavior
- Recent changes that might be related (check git log)

## Output

Provide:
- Root cause analysis
- Specific file(s) and line(s) causing the issue
- Proposed fix with code changes
- Test to verify the fix
