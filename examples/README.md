# Example CLAUDE.md Files

This directory contains real-world examples of `CLAUDE.md` files for different technology stacks. Use these as references when customizing your own project instructions.

## Available Examples

| Example | Tech Stack | Best For |
|---------|------------|----------|
| [typescript-react](typescript-react/CLAUDE.md) | TypeScript, React, Vite | Frontend web applications |
| [python-fastapi](python-fastapi/CLAUDE.md) | Python, FastAPI, SQLAlchemy | Backend APIs and services |
| [node-express](node-express/CLAUDE.md) | Node.js, Express, Prisma | Full-stack JavaScript applications |

## How to Use

1. Choose the example closest to your tech stack
2. Copy the `CLAUDE.md` to your project root
3. Customize the placeholders for your specific project
4. Add project-specific patterns and conventions

## Contributing Examples

Have a `CLAUDE.md` for a different stack? Contributions welcome! Create a new directory with:
- `CLAUDE.md` - The project instructions file
- Brief description of the tech stack and use case
